# aether-core
Modular multimodal HMI and robotics platform integrating voice, eye-tracking, and real-scale visualization.

# Project Aether: Core Framework
**Vector Robotics Official | Phase 1: Active**

## Overview
A multimodal HMI platform integrating Voice (Whisper), Vision (MediaPipe), and Robotics (ROS 2).

## Tech Stack
- **AI:** OpenAI Whisper
- **Automation:** PyAutoGUI
- **Middleware:** ROS 2 Humble
- **Language:** Python 3.10

## Getting Started
1. `git clone https://github.com/vectorroboticsofficial/aether-core.git`
2. `pip install -r requirements.txt`
